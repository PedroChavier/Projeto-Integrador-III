## Descrição do Projeto

O MesclaInvest é um aplicativo mobile desenvolvido na disciplina de Projeto Integrador III do curso de Engenharia de Software.

A aplicação simula uma plataforma digital de investimentos em startups vinculadas ao ecossistema Mescla da PUC-Campinas.

O sistema permite que o usuário:

- Criar conta e realizar login
- Navegar pelo catálogo de startups
- Visualizar informações institucionais e societárias
- Consultar sumário executivo e documentos públicos
- Enviar perguntas aos empreendedores
- Simular compra e venda de tokens
- Acompanhar a valorização dos tokens por meio de dashboard
- Utilizar autenticação multifator (2FA)

Todas as operações financeiras são simuladas, não havendo integração com sistemas bancários reais.
